About the project

Using JS, CSS and a pre-rendered image sequence we created a 360 view image slider, which enables the user to drag and spin the 3D object around its y-axis using simple mouse or touch events.

Rob's tutorial in .net magazine (issue 224) explains how developers can use these simple techniques to create engaging interaction.
